# 行政事務自動化 × AI（表單、SOP、文件彙整）— AI 工具實戰課程

## 課程主題

行政事務自動化 × AI — 表單、SOP、會議紀錄與數據中台實戰課程

## 難易類型

■ 初階課程 □ 中階課程 □ 進階課程

## 辦理目的

進出口貿易業與中小企業常因行政人力不足，導致例行作業（表單填寫、SOP 編製、會議紀錄、報告彙整、資料統計）耗費大量人工時間，且格式不一、版本混亂、資訊分散在各式文件中難以查找與追蹤。本課程協助學員運用 Gemini 自動化產製會議紀錄與行政報告，搭配 Google Canvas 快速建立資訊呈現網站，並透過 Google AI Studio 的 Build 功能打造數據中台，將企業報告匯入後即可自動產出視覺化數據儀表板。課程同時導入 NotebookLM 建構行政知識庫、ChatGPT 自動生成 SOP 與流程圖，以及 Google Forms + Apps Script 串接自動化表單處理流程，全面協助企業建立標準化、自動化、可視化的行政營運體系。

## 辦理目標

- 使用 AI 自動生成標準 SOP 文件（含流程圖與標準作業程序）
- 運用 Gemini 自動化製作會議紀錄與行政報告
- 使用 Google Canvas 建立資訊呈現網站，將行政資訊視覺化呈現
- 透過 Google AI Studio Build 建立數據中台，匯入報告即可呈現視覺化數據
- 運用 NotebookLM 建立行政知識庫，快速查詢與管理企業文件
- 建立表單與資料彙整自動化流程（Google Forms + AI 自動處理）
- 完成一套可驗收的企業行政自動化作業流程

## 培訓對象

- 進出口貿易業者負責人
- 業務助理、行政主管
- 秘書、總務、行政人員
- 貿易公司營運管理人員
- 希望導入 AI 提升行政效率之中小企業主管

## 授課師資條件

1. 具備企業行政管理或數位轉型實務經驗，熟悉行政與文件管理作業。
2. 熟悉 Gemini、ChatGPT、NotebookLM 等 AI 工具於行政自動化之應用。
3. 曾運用 Google AI Studio 或 Google Canvas 進行資料視覺化或應用建置之教學經驗。

## 授課時數

6 小時

## 課程大綱

### 單元一、行政痛點解析 × AI 流程化概念（30 分鐘）

**學習目標：** 了解企業行政作業常見瓶頸，認識 AI 在行政自動化中的應用場景與工具架構。

| 項目     | 說明                |
| -------- | ------------------- |
| 時間     | 30 分鐘             |
| 教學方式 | 講解 + 情境案例分享 |

**教學內容：**

- 進出口貿易業行政痛點盤點
  - SOP 文件四散、版本不統一，新人無所適從
  - 會議紀錄手打耗時，且常遺漏重點
  - 行政報告（週報 / 月報 / 出貨統計）手動彙整費時
  - 表單資料零散（紙本 / Email / Google Form），缺乏統一匯整
  - 資料查找困難，企業知識無法有效傳承
- AI 行政自動化的四大應用層次
  - **生成層**：AI 自動產出 SOP、會議紀錄、報告文案
  - **彙整層**：AI 將分散資料統一整理、摘要、分類
  - **呈現層**：Canvas 網站 / AI Studio 數據中台視覺化
  - **知識層**：NotebookLM 行政知識庫，隨問隨查
- 情境案例：一家貿易公司如何用 AI 將每週行政報告從 4 小時縮短至 20 分鐘
- 本日課程學習地圖與工具全覽

---

### 單元二、AI 製作 SOP 與流程圖（60 分鐘）

**學習目標：** 學會使用 AI 快速產出企業標準作業程序（SOP）文件與視覺化流程圖。

| 項目     | 說明                   |
| -------- | ---------------------- |
| 時間     | 60 分鐘                |
| 教學方式 | 講解 + 示範 + 學員實作 |
| 使用工具 | ChatGPT / Gemini       |

**教學內容：**

- SOP 自動化生成方法論
  - Prompt 設計：描述作業流程 → AI 產出結構化 SOP
  - 輸出格式：步驟編號、負責人、注意事項、異常處理
  - 範例：出貨作業 SOP、客戶報價作業 SOP、樣品寄送 SOP
- AI 生成視覺化流程圖
  - AI 將文字 SOP 自動轉換為流程圖（使用 Mermaid / draw.io 語法）
  - 包含決策節點、並行流程、例外處理分支
  - 範例：進口報關流程圖、客訴處理流程圖
- SOP 版本管理與更新
  - AI 比對新舊 SOP 差異，自動標註修改處
  - 建立 SOP 文件命名與歸檔規範

**學員實作：**

- 選擇一項自家企業行政流程，使用 AI 產出完整 SOP 文件
- 將 SOP 轉換為流程圖（使用 AI 生成 Mermaid 語法或圖片）

---

### 單元三、Gemini 自動生成會議紀錄與行政報告（60 分鐘）

**學習目標：** 運用 Gemini 將會議錄音 / 逐字稿自動轉為結構化會議紀錄，並產出行政報告。

| 項目     | 說明                   |
| -------- | ---------------------- |
| 時間     | 60 分鐘                |
| 教學方式 | 講解 + 示範 + 學員實作 |
| 使用工具 | Gemini / Google Docs   |

**教學內容：**

- Gemini 會議紀錄自動化
  - 匯入會議錄音檔 / 逐字稿，Gemini 自動產出結構化紀錄
  - 輸出格式：會議主題、與會人員、討論事項、決議事項、待辦追蹤（含負責人與期限）
  - Prompt 技巧：指定紀錄格式、摘要層級、重點標註
  - 多語系支援：中英文會議自動轉錄與摘要
- Gemini 行政報告自動產製
  - 週報 / 月報自動生成：匯入原始資料 → AI 自動摘要與分析
  - 出貨統計報告：匯入出貨明細 → 自動產出數據摘要與異常標註
  - 客戶拜訪報告：依據拜訪紀錄 → 自動生成報告含建議行動
- 進階應用：串接 Google Docs
  - Gemini 在 Google Docs 中直接生成、改寫、摘要
  - 建立報告模板，一鍵套用格式

**學員實作：**

- 使用 Gemini 將範例會議逐字稿轉為結構化會議紀錄
- 使用 Gemini 匯入範例數據，自動產出一份行政週報

---

### 單元四、工具操作 — Google Canvas 資訊網站 + AI Studio 數據中台（120 分鐘）

**學習目標：** 使用 Google Canvas 建立資訊呈現網站，並透過 Google AI Studio Build 打造數據中台，匯入報告即可呈現視覺化數據。

| 項目     | 說明                                                   |
| -------- | ------------------------------------------------------ |
| 時間     | 120 分鐘                                               |
| 教學方式 | 講師示範 + 學員實作                                    |
| 使用工具 | Google Canvas / Google AI Studio Build / Google Sheets |

**講師示範：**

1. **Google Canvas 建立行政資訊呈現網站（40 分鐘）**
   - Canvas 基礎介紹：用 AI 對話式生成互動網頁
   - 建立企業內部資訊看板網站
     - 公司 SOP 查詢頁面：依部門 / 流程分類
     - 會議紀錄彙整頁面：依日期 / 專案分類呈現
     - 公告與待辦事項追蹤頁面
   - 設計技巧：版面配置、配色、互動元素設定
   - 範例展示：一個貿易公司的內部行政入口網站

2. **Google AI Studio Build 打造數據中台（50 分鐘）**
   - AI Studio Build 功能介紹：用自然語言建構數據應用
   - 建立數據中台的步驟
     - 匯入資料來源：Google Sheets / CSV 報告檔案
     - 設定數據結構與欄位對應
     - 用自然語言指令建立視覺化圖表
   - 視覺化儀表板實作
     - 出貨金額趨勢圖（折線圖 / 長條圖）
     - 客戶地區分布熱力圖
     - 產品類別銷售佔比圓餅圖
     - 月度 KPI 達成率儀表板
   - 互動功能：篩選器、日期區間、下鑽分析
   - 將報告匯入後自動更新數據呈現

3. **兩者整合應用展示（10 分鐘）**
   - Canvas 資訊網站嵌入 AI Studio 數據中台連結
   - 建立「一站式行政管理入口」：SOP + 會議紀錄 + 數據報表

**學員實作（20 分鐘）：**

- 使用 Google Canvas 建立一個簡易的企業資訊看板（含 SOP 查詢 + 公告頁面）
- 使用 Google AI Studio Build 匯入範例出貨報告，產出 2～3 組視覺化圖表
- 確認數據中台可正確呈現匯入資料的視覺化結果

### 單元五、NotebookLM 行政知識庫 + 表單自動化演練（60 分鐘）

**學習目標：** 建立企業行政知識庫供快速查詢，並整合表單與資料彙整自動化流程。

| 項目     | 說明                                                |
| -------- | --------------------------------------------------- |
| 時間     | 60 分鐘                                             |
| 教學方式 | 講解 + 示範 + 分組演練                              |
| 使用工具 | NotebookLM / Google Forms / Google Sheets / ChatGPT |

**教學內容：**

1. **NotebookLM 建立行政知識庫（30 分鐘）**
   - NotebookLM 功能介紹與操作
   - 匯入企業文件：SOP、合約範本、公司規章、歷史報告
   - AI 對話式查詢：「出口報關需要哪些文件？」「客訴處理 SOP 是什麼？」
   - 自動交叉引用：AI 從多份文件中整合答案並標註出處
   - 企業應用：新人 Onboarding 快速自學、跨部門知識共享

2. **表單與資料彙整自動化（30 分鐘）**
   - Google Forms 建立標準化表單
     - 請假申請表、出差申請表、費用報支表
     - 客戶拜訪回報表、樣品需求表
   - 表單回覆自動彙整至 Google Sheets
   - ChatGPT 輔助批次處理：自動分類、摘要、異常標註
   - 進階串接：表單提交後自動通知主管（Email / LINE Notify）
   - 範例：每月自動彙整所有出差報告，產出月度出差統計摘要

**分組演練：**

- **場景 A：新人到職第一天**
  - 任務：用 NotebookLM 建立行政知識庫，讓新人可直接查詢公司 SOP、請假規定、報銷流程
  - 工具：NotebookLM + 企業文件範例

- **場景 B：月度行政報告自動化**
  - 任務：設計一套流程：各部門填寫 Google Form 週報 → 自動彙整至 Sheets → AI 產出月度統整報告
  - 工具：Google Forms + Google Sheets + Gemini

- **場景 C：貿易文件快速查找**
  - 任務：匯入進出口相關法規、合約範本、報關流程文件，建立可對話查詢的知識庫
  - 工具：NotebookLM
